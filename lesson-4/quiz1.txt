Q2.Which is the most succinct query to return all documents from the sample_training.inspections collection where the inspection date is either "Feb 20 2015", or "Feb 21 2015" and the company
 is not part of the "Cigarette Retail Dealer - 127" sector?

 ANS:
 1. db.inspections.find(
   { "$or": [ { "date": "Feb 20 2015" },
             { "date": "Feb 21 2015" } ],
    "sector": { "$ne": "Cigarette Retail Dealer - 127" }}).pretty()

Q3.What are some of the uses for the $ sign in MQL?

Ans:
$ denotes an operator.
$ signifies that you are looking at the value of that field rather than the field name.

Q5.Which of the following queries will return all listings that have "Free parking on premises", "Air conditioning", and "Wifi" as part of their 
amenities, and have at least 2 bedrooms in the sample_airbnb.listingsAndReviews collection? 

 ANS:
 2. db.listingsAndReviews.find(
  { "amenities":
      { "$all": [ "Free parking on premises", "Wifi", "Air
        conditioning" ] }, "bedrooms": { "$gte":  2 } } ).pretty()

Q6.Which of the following queries will return only the names of companies from the 
sample_training.companies collection that had exactly 8 funding rounds?

ANS:
3. db.companies.find({ "funding_rounds": { "$size": 8 } },
                  { "name": 1, "_id": 0 })
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Q7.Which of the following queries will return the names and addresses of all listings from the sample_airbnb.listingsAndReviews 
collection where the first amenity in the list is "Internet"?

ANS:
3. db.listingsAndReviews.find({ "amenities.0": "Internet" },
                           { "name": 1, "address": 1 }).pretty()
